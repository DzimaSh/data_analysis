# Лабараторная праца №6: Рэкурэнтныя нейронныя сеткі (RNN)

Гэты каталог утрымлівае матэрыялы для Лабараторнай працы №6.

## 🎯 Мэта працы
Мэта гэтай лабараторнай працы — вывучэнне мадэлявання паслядоўнасцей і апрацоўкі прыроднай мовы (NLP) з выкарыстаннем розных тыпаў рэкурэнтных нейронавых сетак (RNN).

## 📚 Асноўныя тэмы для вывучэння
1.  **Простая рэкурэнтная сетка (Vanilla RNN):** архітэктура, праблема знікнення/выбуху градыентаў (vanishing/exploding gradients).
2.  **Мадэлі з памяццю (LSTM і GRU):** праца варот (gates), захаванне доўгатэрміновай памяці ў паслядоўнасцях.
3.  **Вэктарнае прадстаўленне слоў (Word Embeddings):** выкарыстанне пласта `Embedding`, канцэпцыі Word2Vec і GloVe.
4.  **Практычная задача:** класіфікацыя тэкстаў (напрыклад, аналіз танальнасці водгукаў IMDB ці навінавых загалоўкаў) і вымярэнне якасці прагнозаў (Loss, Accuracy, F1-Score).

*Заўвага: Рэалізацыя коду і практычныя заданні будуць дададзены ў межах выканання адпаведнага этапу курса.*
