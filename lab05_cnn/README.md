# Лабараторная праца №5: Свёртачныя нейронныя сеткі (CNN)

Гэты каталог утрымлівае матэрыялы для Лабараторнай працы №5.

## 🎯 Мэта працы
Мэта гэтай лабараторнай працы — азнаямленне з асновамі глыбокага навучання (Deep Learning) і архітэктурай свёртачных нейронавых сетак (CNN) для камп'ютарнага зроку (Computer Vision) і распазнавання выяў.

## 📚 Асноўныя тэмы для вывучэння
1.  **Свёртачныя пласты (Convolutional Layers):** аперацыя згорткі, ядры (фільтры), атрыманне карт прыкмет (feature maps).
2.  **Пласты субдыскрэтызацыі (Pooling Layers):** MaxPooling і AveragePooling для зніжэння размернасці даных.
3.  **Поўназвязныя пласты (Fully Connected Layers):** класіфікацыя на аснове згенераваных прыкмет.
4.  **Функцыі актывацыі (ReLU, Softmax) і метады аптымізацыі (Adam, SGD).**
5.  **Паўнавартасны цыкл навучання:** навучанне CNN на класічным датасеце выяў (напрыклад, MNIST, CIFAR-10 ці Fashion-MNIST), ацэнка якасці і аналіз матрыцы памылак (Confusion Matrix).

*Заўвага: Рэалізацыя коду і практычныя заданні будуць дададзены ў межах выканання адпаведнага этапу курса.*
