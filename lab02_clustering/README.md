# Лабараторная праца №2: Кластэрызацыя даных і зніжэнне размернасці (Clustering & Dimensionality Reduction)

Справаздача і аналітычныя вынікі па выкананні Лабараторнай працы №2.

---

## 🎯 Мэта працы
Вывучэнне і практычнае прымяненне метадаў машыннага навучання без настаўніка (unsupervised learning) для кластэрызацыі таблічных даных на прыкладзе набору даных **Wine** з бібліятэкі `scikit-learn`. Задача ўключае падрыхтоўку даных, зніжэнне размернасці (PCA, t-SNE), рэалізацыю і падбор аптымальных параметраў для трох розных алгарытмаў кластэрызацыі (K-Means, агламератыўная іерархічная кластэрызацыя, DBSCAN) і ацэнку іх якасці паводле ўнутраных метрык.

---

## 📊 Выкарыстаны датасет і перадапрацоўка

Для аналізу выкарыстоўваецца стандартны набор даных **Wine** (хімічны склад вінаў трох італьянскіх вытворцаў):
*   **Памернасць:** 178 аб'ектаў, 13 лікавых прыкмет.
*   **Прыкметы:** `alcohol`, `malic_acid`, `ash`, `alcalinity_of_ash`, `magnesium`, `total_phenols`, `flavanoids`, `nonflavanoid_phenols`, `proanthocyanins`, `color_intensity`, `hue`, `od280/od315_of_diluted_wines`, `proline`.
*   **Выключэнне мэтавай зменнай:** Для забеспячэння чыстага эксперыменту без настаўніка слупок класаў (`target`) быў цалкам выдалены на самым пачатку працы. Ён выкарыстоўваўся толькі для фінальнай візуальнай ацэнкі адпаведнасці.
*   **Ачыстка:** Праведзена праверка на пропускі, якая пацвердзіла поўную цэласнасць даных (0 пропускаў).
*   **Маштабаванне:** Усе 13 прыкмет былі стандартызаваны з дапамогай `StandardScaler` (прывядзенне да $\mu=0$ і $\sigma=1$), паколькі алгарытмы K-Means, DBSCAN і іерархічная кластэрызацыя заснаваны на метрыцы адлегласці і вельмі адчувальныя да розных маштабаў прыкмет.

## 📉 Зніжэнне размернасці (PCA & t-SNE)

Каб візуалізаваць размеркаванне даных у 2D і 3D прасторах, прыменены два метады:
1.  **PCA (Метад галоўных кампанент):**
    *   Першая галоўная кампанента (ПК1) тлумачыць **36.20%** дысперсіі.
    *   Другая галоўная кампанента (ПК2) тлумачыць **19.21%** дысперсіі.
    *   Сумарна дзве кампаненты (2D) захоўваюць **55.41%** агульнай інфармацыі, а тры кампаненты (3D) — **66.53%**.
2.  **t-SNE:**
    *   Наладжаны з параметрам `perplexity=30`. Ён выразна дэманструе групоўку пунктаў у тры лакальныя кластары, хоць межы паміж двума з іх з'яўляюцца даволі шчыльнымі і блізкімі.

---

## 🛠️ Рэалізацыя і падбор параметраў алгарытмаў

### 1. K-Means (К-сярэдніх)
*   **Падбор k:** Сістэматычна прааналізаваны значэнні ад $k=2$ да $10$.
    *   *Метад локця (Elbow Method):* Графік WCSS (інерцыі) паказвае выразны згін ("локаць") у кропцы **k = 3**.
    *   *Сілуэтны аналіз (Silhouette Analysis):* Максімальнае значэнне каэфіцыента сілуэту **0.2849** дасягаецца пры **k = 3**.
*   **Вынік:** Аптымальная колькасць кластараў прынята роўнай 3. Размеркаванне аб'ектаў: Кластэр 0 — 62 кропкі, Кластэр 1 — 65 кропак, Кластэр 2 — 51 кропка.

### 2. Агламератыўная іерархічная кластэрызацыя
*   **Налада:** Выкарыстаны метад сувязі Уорда (`ward linkage`) і эўклідава адлегласць.
*   **Падбор k:** Пабудавана дэндаграма. Гарызантальнае адсячэнне на вышыні адлегласці $10.5$ падзяляе дрэва сувязей роўна на **3 кластары**, што цалкам узгадняецца з вынікамі K-Means.
*   **Вынік:** Мадэль навучана для 3 кластараў (сілуэтны бал — **0.2774**).

### 3. DBSCAN (Шчыльнасная кластэрызацыя)
*   **Падбор параметраў:**
    *   *Eps:* З дапамогай графіка $k$-адлегласці (пры $k=4$) знойдзены арыенціровачны згін у дыяпазоне $2.3$ - $2.5$.
    *   *Grid Sweep:* Праведзены сеткавы пошук па `eps` $[2.0 \dots 3.0]$ і `min_samples` $[3 \dots 10]$.
*   **Вынік:** Аптымальнымі выбраны параметры **eps = 2.40**, **min_samples = 3**.
    *   Алгарытм сфармаваў **2 кластары** і вылучыў **28 шумавых пунктаў (15.73%)**.
    *   *Фізічны сэнс:* DBSCAN аб'яднаў два блізкія і шчыльныя гатункі вінаграда ў адзін буйны кластар з-за іх высокай шчыльнасці і блікага перакрыцця ў 13D-прасторы, вылучыўшы трэці гатунак у асобны кластар, а памежныя кропкі пазначыў як шум.

## 🏆 Ацэнка і параўнанне якасці кластэрызацыі

Якасць кластэрызацыі ацэньвалася з дапамогай трох унутраных метрык без настаўніка:

| Метрыка | K-Means | Hierarchical (Ward) | DBSCAN (eps=2.4, min_samples=3) |
| :--- | :---: | :---: | :---: |
| **Колькасць кластараў** | 3 | 3 | 2 |
| **Колькасць шуму (выкідаў)** | 0 | 0 | 28 (15.73%) |
| **Каэфіцыент сілуэту (Silhouette)** $\uparrow$ | **0.2849** | 0.2774 | 0.2144 |
| **Індэкс Калінскі-Харабаса (CH)** $\uparrow$ | **70.94** | 67.65 | 32.65 |
| **Індэкс Дэвіса-Боўлдзіна (DB)** $\downarrow$ | **1.3892** | 1.4186 | 3.2177 |

*Заўвага: Для метрык Silhouette і Calinski-Harabasz лепшымі з'яўляюцца больш высокія значэнні ($\uparrow$), для Davies-Bouldin — больш нізкія ($\downarrow$).*

## 🔍 Асноўныя аналітычныя высновы

1.  **Лідар па якасці:** Алгарытм **K-Means** паказаў найлепшыя вынікі па ўсіх трох метрыках: Silhouette (**0.2849**), Calinski-Harabasz (**70.94**) і Davies-Bouldin (**1.3892**). Гэта тлумачыцца тым, што кластары ў стандартызаванай прасторы Wine маюць даволі выпуклую сферычную форму, што з'яўляецца ідэальным для цэнтрыядных метадаў.
2.  **Эфектыўнасць іерархічнага метаду:** Агламератыўная кластэрызацыя з метадам сувязі Уорда дэманструе вынікі, вельмі блізкія да K-Means (Silhouette = **0.2774**), што пацвярджае стабільнасць знойдзенай структуры з 3-х кластараў.
3.  **Асаблівасці DBSCAN:** Шчыльнасны алгарытм DBSCAN вельмі карысны для выяўлення выкідаў (выдзелена 28 нестабільных кропак на межах), аднак з-за высокай размернасці і рознай шчыльнасці ён аб'ядноўвае два гатункі ў адзін, што прыводзіць да зніжэння фармальных метрык якасці.
4.  **Супастаўленне з рэальнымі гатункамі:** Супастаўленне вынікаў кластэрызацыі K-Means з рэальнымі класамі (`y_true`) на 2D-праекцыі PCA паказала практычна ідэальнае супадзенне. Гэта сведчыць аб тым, што прыкметы хімічнага складу маюць моцную прыродную групавую структуру, якую алгарытмы машыннага навучання здольныя цалкам аднавіць без выкарыстання мэтавых пазнак.
