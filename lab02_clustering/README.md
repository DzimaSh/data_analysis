# Лабараторная праца №2: Кластэрызацыя даных (Clustering)

Гэты каталог утрымлівае матэрыялы для Лабараторнай працы №2.

## 🎯 Мэта працы
Мэта гэтай лабараторнай працы — вывучэнне і практычнае прымяненне алгарытмаў машыннага навучання без настаўніка (unsupervised learning) для задачы кластэрызацыі даных.

## 📚 Асноўныя тэмы для вывучэння
1.  **Алгарытм K-Means (К-сярэдніх):** прынцып працы, выбар аптымальнай колькасці кластараў метадам "локця" (elbow method) і сілуэтным аналізам (silhouette score).
2.  **DBSCAN (шчыльнасны метад):** пошук кластараў адвольнай формы і фільтрацыя шуму/выкідаў.
3.  **Іерархічная кластэрызацыя (Hierarchical Clustering):** пастраенне дэндаграм і агламератыўны падыход.
4.  **Метрыкі ацэнкі якасці кластэрызацыі:** Silhouette Coefficient, Adjusted Rand Index (ARI), Homogeneity/Completeness.

*Заўвага: Рэалізацыя коду і практычныя заданні будуць дададзены ў межах выканання адпаведнага этапу курса.*
