# Лабараторная праца №7: Аналіз часавых радаў з Prophet і RNN (Time Series)

Гэты каталог утрымлівае матэрыялы для Лабараторнай працы №7.

## 🎯 Мэта працы
Мэта гэтай лабараторнай працы — вывучэнне спецыфікі аналізу, апрацоўкі і прагназавання паслядоўнасцей, якія залежаць ад часу (часавых радаў).

## 📚 Асноўныя тэмы для вывучэння
1.  **Кампаненты часавага рада:** трэнд, сезоннасць, цыклічнасць і белы шум. Стацыянарнасць часавага рада.
2.  **Мадэляванне з Facebook Prophet:** адытыўныя і мультыплікатыўныя мадэлі, улік святаў, трэндавых кропак (changepoints) і сезоннасці.
3.  **Глыбокае навучанне для часавых радаў:** выкарыстанне LSTM/GRU сетак для шматкрокавага прагназавання (multistep forecasting) будучых значэнняў.
4.  **Ацэнка якасці прагнозаў:** MAE, RMSE, MAPE (Mean Absolute Percentage Error).

*Заўвага: Рэалізацыя коду і практычныя заданні будуць дададзены ў межах выканання адпаведнага этапу курса.*
