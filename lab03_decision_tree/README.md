# Лабараторная праца №3: Дрэвы рашэнняў (Decision Trees)

Гэты каталог утрымлівае матэрыялы для Лабараторнай працы №3.

## 🎯 Мэта працы
Мэта гэтай лабараторнай працы — азнаямленне з алгарытмам дрэў рашэнняў (Decision Trees) для задач класіфікацыі і рэгрэсіі, вывучэнне крытэрыяў інфарматыўнасці і метадаў барацьбы з перанавучаннем мадэлі.

## 📚 Асноўныя тэмы для вывучэння
1.  **Крытэрыі расшчаплення (Splitting Criteria):** Энтрапія Шэнана, індэкс Джыні (Gini impurity) для класіфікацыі, і сярэдняквадратычная памылка (MSE/MAE) для рэгрэсіі.
2.  **Перанавучанне (Overfitting):** прычыны і прыкметы перанавучання ў дрэвах.
3.  **Рэгулярызацыя дрэва рашэнняў:** абмежаванне максімальнай глыбіні (`max_depth`), мінімальнай колькасці аб'ектаў у лісце (`min_samples_leaf`), стрыжка дрэў (pruning).
4.  **Візуалізацыя дрэва рашэнняў:** выкарыстанне `plot_tree` альбо Graphviz для графічнага прадстаўлення структуры прыняцця рашэнняў мадэллю.

*Заўвага: Рэалізацыя коду і практычныя заданні будуць дададзены ў межах выканання адпаведнага этапу курса.*
