# Лабараторная праца №4: Ансамблі мадэляў (Ensemble Methods)

Гэты каталог утрымлівае матэрыялы для Лабараторнай працы №4.

## 🎯 Мэта працы
Мэта гэтай лабараторнай працы — вывучэнне магутных метадаў аб'яднання некалькіх базавых алгарытмаў у адзіны ансамбль для павышэння якасці прагназавання і зніжэння хібы мадэлі.

## 📚 Асноўныя тэмы для вывучэння
1.  **Бэгінг (Bagging) і Выпадковы лес (Random Forest):** асерадненне прагнозаў незалежных дрэў рашэнняў, Out-of-Bag (OOB) памылка.
2.  **Бустынг (Boosting):** паслядоўная пабудова ансамбля, дзе кожная наступная мадэль выпраўляе памылкі папярэдніх.
3.  **Сучасныя бібліятэкі градыентнага бустынгу:** азнаямленне з алгарытмамі AdaBoost, Gradient Boosting, а таксама высокапрадукцыйнымі фрэймворкамі XGBoost, LightGBM і CatBoost.
4.  **Ацэнка важнасці прыкмет (Feature Importance):** інтэрпрэтацыя вынікаў працы ансамбля мадэляў.

*Заўвага: Рэалізацыя коду і практычныя заданні будуць дададзены ў межах выканання адпаведнага этапу курса.*
