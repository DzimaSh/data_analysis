# Каталог даных (Data Directory)

Гэты каталог прызначаны для захоўвання файлаў даных Лабараторнай працы №1.

## ℹ️ Крыніца даных
У гэтай лабараторнай працы мы выкарыстоўваем класічны набор даных **Titanic**, які загружаецца **праграмна** непасрэдна з рэпазіторыя `seaborn` пры дапамозе функцыі:
```python
import seaborn as sns
df = sns.load_dataset('titanic')
```
Таму няма неабходнасці ўручную спампоўваць і захоўваць файл даных `.csv` у гэтай тэчцы. Увесь працэс загрузкі цалкам аўтаматызаваны ўнутры Jupyter Notebook.
