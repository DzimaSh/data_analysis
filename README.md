# Лабараторныя працы па аналізе даных (Data Analysis Labs)

Гэты рэпазіторый змяшчае поўны набор матэрыялаў і кодаў для навучальнага курса, які складаецца з 7 лабараторных прац па аналізе даных і машынным навучанні. 

Усе файлы дакументацыі, справаздачы, каментарыі і апісанні выкананы **выключна на беларускай мове**.

## 📂 Структура рэпазіторыя

Рэпазіторый арганізаваны наступным чынам:

*   **`lab01_eda_predictive/`** (Поўная рэалізацыя) — Разведачны аналіз даных (EDA), ачыстка даных, генерацыя прыкмет і пабудова першай прагназавальнай мадэлі (на прыкладзе класічнага датасета *Titanic*).
*   **`lab02_clustering/`** (Толькі структура) — Кластэрызацыя даных (K-Means, DBSCAN, Hierarchical Clustering).
*   **`lab03_decision_tree/`** (Толькі структура) — Дрэвы рашэнняў для задач класіфікацыі і рэгрэсіі.
*   **`lab04_ensembles/`** (Толькі структура) — Ансамблі мадэляў (Random Forest, Gradient Boosting, XGBoost).
*   **`lab05_cnn/`** (Толькі структура) — Свёртачныя нейронныя сеткі (CNN) для апрацоўкі выяў.
*   **`lab06_rnn/`** (Толькі структура) — Рэкурэнтныя нейронныя сеткі (RNN) для аналізу паслядоўнасцей і тэкстаў.
*   **`lab07_time_series_prophet/`** (Толькі структура) — Аналіз часавых радаў з выкарыстаннем Facebook Prophet і RNN.

## 🛠️ Патрабаванні і ўсталёўка

Для працы з праектамі вам спатрэбіцца Python версіі **3.10** альбо больш новай, а таксама ўсталяваны менеджар пакетаў `pip`.

### 1. Кланіраванне рэпазіторыя
```bash
git clone <url_рэпазіторыя>
cd data_analysis
```

### 2. Усталёўка залежнасцей
Рэкамендуецца выкарыстоўваць глабальнае ці сістэмнае асяроддзе без `venv` (у адпаведнасці з наладамі). Усталюйце неабходныя бібліятэкі праз `requirements.txt`:
```bash
pip install -r requirements.txt
```

## 🚀 Запуск лабараторных прац

Усе лабараторныя працы, якія змяшчаюць працоўны код, прадстаўлены ў выглядзе інтэрактыўных наўтбукаў Jupyter (`.ipynb`).

Каб запусціць Jupyter Notebook лакальна:
```bash
python -m jupyter notebook
```
Пасля гэтага адкрыйце вэб-інтэрфейс у браўзеры і перайдзіце да патрэбнага файла (напрыклад, `lab01_eda_predictive/eda_and_modeling.ipynb`).

Для праграмнага запуску і перавылічэння ўсіх ячэек ноўтбука з захаваннем вынікаў наўпрост з каманднага радка выкарыстоўвайце `nbconvert`:
```bash
python -m jupyter nbconvert --to notebook --execute --inplace lab01_eda_predictive/eda_and_modeling.ipynb
```
